# Changelog

## 0.1.0

First release.

- Standalone document tree in its own activity-bar view, beside the built-in Explorer.
- Emoji folder icons (bundled Twemoji), set per exact path via right-click.
- Display-name aliases without renaming the real file.
- Manual drag-to-reorder, persisted per folder.
- Hide doc-irrelevant files/folders.
- Git status decorations (via `resourceUri`), inheriting the active file-icon theme for non-emoji items.
- Toolbar: new file / new folder / expand all / collapse all / refresh.
- Context menu: rename, copy, paste, duplicate, copy path, copy relative path, set icon (folders), set alias, hide, reveal in Finder, delete.
- Keyboard shortcuts inside the view: F2 rename, ⌘C/⌘V copy/paste, ⌘D duplicate, ⌘⌫ delete, ⌘⌥C copy path.
